# AI Life Manager Documentation

Welcome to the AI Life Manager documentation.

## Overview
This is a privacy-first, AI-powered personal assistant that helps you manage daily tasks, track mood, and plan your day.

## Architecture
- **Backend**: FastAPI (Python 3.12)
- **Database**: PostgreSQL 16
- **Caching**: Redis
- **AI**: LangChain + OpenAI GPT-4o-mini
